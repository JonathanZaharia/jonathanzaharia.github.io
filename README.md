# Boots-Wheel-of-Fortune-Joke-
Birthday Wheel of Boots — a silly Wheel of Fortune HTML spinwheel where the winning prize is a Jim Green boot. A tiny, ridiculous HTML/CSS/JS Wheel of Fortune for a Jim Green boots birthday gag. Spin the wheel, watch the confetti, and if Lady Luck smiles you win: Boot to Buy — the very boot you must purchase (or gift).
