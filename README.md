#Wheel of Boots — Jim Green Birthday Joke
A tiny, ridiculous HTML/CSS/JS Wheel of Fortune for a Jim Green boots birthday gag. Spin the wheel, watch the confetti, and if Lady Luck smiles you win: Boot to Buy — the very boot you must purchase (or gift).

# How to use

Open index.html in any browser.
Click Spin.
Celebrate (or ruefully checkout the winning boot link).

# Features

Single-file, no dependencies (just index.html, optional styles.css and script.js)
Randomized spins with simple easing animation
Big confetti and a cheeky “You won: Boot to Buy” modal for winners
Suggested files
index.html — wheel UI & markup
styles.css — quick styling (mobile-friendly)
script.js — spin logic + winner handling
